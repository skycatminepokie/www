# Mclogs Uploader

A small Discord bot I made to upload attachments to <https://mclo.gs/>. Helpful when you are on mobile and can't see
attachments, or on desktop and an attachment is too big. Only works for text. See [TOS](./tos.md) and
[Privacy Policy](./privacy.md) (don't worry, they're simple).
